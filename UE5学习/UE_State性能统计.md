# UE_State性能统计


### 统计数据信息

| 名称             | 意味着什么                                                                                                                                     |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Game**       | 游戏主线程（Game Thread）的处理时间，包括游戏逻辑、Tick、AI、物理等。若这个值很高，说明逻辑耗时多。([RIRAGON（リラゴン）][1])                                                            |
| **Draw**       | 渲染线程（Render Thread）处理时间，用于准备渲染命令等，是 CPU 上的图形准备时间。([RIRAGON（リラゴン）][1])                                                                     |
| **RHIT**       | RHI Thread（RHI 线程）的处理时间 —— 又叫 **Render Hardware Interface Thread**，就是负责把渲染命令真正下发到 GPU 这块的线程。若这值高，可能是大量渲染命令或驱动/硬件接口较慢。([RIRAGON（リラゴン）][1]) |
| **GPU (Time)** | GPU 实际执行渲染任务所花的时间，是显卡做绘制、Lighting、Post Process 等工作的总时间。([RIRAGON（リラゴン）][1])                                                               |
| **Input**      | 输入处理时间 —— 有时候会显示为用户输入事件处理所需的时间（键盘/鼠标/手柄等），高值可能来自 Input 事件绑定处理或某些延迟统计机制。([よっしーの100チャレンジ][2])                                               |
