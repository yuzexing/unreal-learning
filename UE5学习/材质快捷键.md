# 材质快捷键

### 向量

| 快捷键                                        | 作用                   | 说明             |
| ------------------------------------------ | -------------------- | -------------- |
| **1 + 左键**                                 | Constant (标量常数)      | 1个数值节点（float）  |
| **2 + 左键**                                 | Constant2Vector      | 2维向量           |
| **3 + 左键**                                 | Constant3Vector      | 3维向量（如 RGB 颜色） |
| **4 + 左键**                                 | Constant4Vector      | 4维向量（RGBA）     |
| **S + 左键**                                 | **Scalar Parameter** | 直接创建可调的标量参数    |
| **V + 左键**                                 | **Vector Parameter** | 创建可调的向量/颜色参数   |

### 运算

| 快捷键              | 节点            |
| ---------------- | ------------- |
| **A + 左键**       | Add（加法）       |
| **M + 左键**       | Multiply（乘法）  |
| **D + 左键**       | Divide（除法）    |
| **L + 左键**       | Lerp（线性插值）    |
| **O + 左键**       | OneMinus（1–x） |


### 纹理

| 快捷键               | 节点                       |
| ----------------- | ------------------------ |
| **T + 左键**        | Texture Sample（贴图采样）     |
| **U + 左键**        | Texture Coordinate（UV坐标） |
| **P + 左键**        | Panner（UV滚动） |
