# GASTag


### Tag的添加方式

1. 编辑器中，通过GameplayTagManager添加
2. GameplayTagTableList中添加Tags的Data Table
3. 通过UGameplayTagManager，添加Native的GameplayTags

### Tag的作用

1. 通过Tag关联输入事件
2. 通过tag关联属性
3. 通过Tag关联UI
4. 通过Tag关联GA
