# TCP和UDP

TCP和UDP都属于传输层协议，

### TCP
特点
1. 面向连接
2. 可靠传输
3. 面向字节流
4. 资源开销大


### UDP
特点：
1. 面向无连接
2. 不可靠传输
3. 面向数据报
4. 延迟低
