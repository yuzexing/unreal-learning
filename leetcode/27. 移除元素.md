# 27. 移除元素

### 初次提交
```
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int i = 0;
        int j = nums.size() - 1;
        while (i <= j) {
            if (nums[i] == val) {
                swap(nums[i], nums[j]);
                j --;
                continue;
            }
            i ++;
        }
        return i;
    }
};
```
